# 亚马逊订单转发票（纯前端）

一个本地网页工具：把 Seller Central 订单详情页复制出来的文本粘贴进去，自动生成发票预览，并通过浏览器打印导出为 PDF。

## 使用方式

1. 直接双击打开 `index.html`（或使用任意静态服务器）。
2. 在左侧粘贴完整的订单详情文本。
3. 点击“解析并生成发票”。
4. 检查右侧发票预览，必要时修改卖家信息。
5. 点击“导出 PDF”，在浏览器打印窗口选择“另存为 PDF”。

## 当前已解析字段

- 订单号、购买日期、配送截止日期、销售渠道、付款方式
- 买家姓名、买家地址、联系人
- 商品名称、ASIN、SKU、订单商品编号、数量
- 商品总计（含税）、VAT 总计、销售净收益（不含税）
- 税务信息（收税模型、Legislation、收税责任方）

## 说明

- 此版本按你提供的文本格式做了规则解析。
- 如果后续有新的订单页面格式差异，可继续补充解析规则。
